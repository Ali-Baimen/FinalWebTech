# TV Shows

TV Lite is a responsive front-end website created as part of a coursework project.  
It showcases a collection of TV shows with a modern dark/light theme, interactive UI components, and form handling using JavaScript and jQuery.

---

## Project Overview

The website contains multiple pages:
- **Home (`index.html`)** – Landing page introducing the site and featured content.
- **Shows (`shows.html`)** – Main catalog displaying cards of popular and trending TV shows.
- **About (`about.html`)** – Information about the creators and the project.
- **Support (`support.html`)** – Contact form for visitors to send messages.
- **Sign In / Sign Up (`signin.html`, `signup.html`)** – Basic login and registration forms with validation.
- **Profile (`profile.html`)** – Simple user profile layout.
- **Terms & Conditions (`terms.html`)** – Legal and privacy information linked from the footer.

---

## Features

- **Responsive Design**  
  Built with HTML5, CSS3, Bootstrap 5, and custom media queries for full mobile and desktop compatibility.

- **Theme Toggle (Light / Dark Mode)**  
  Implemented via a custom switch using CSS variables and jQuery to persist style changes.

- **Dynamic Content**  
  - Search and live suggestions on the “Shows” page.  
  - Animated counters on the “About” page.  
  - FAQ accordion component using JavaScript transitions.  
  - Real-time date/time and greetings scripts.  
  - Pop-up subscription window with interactive behavior.

- **Form Validation**  
  Sign Up and Support forms include input validation with contextual messages.

- **jQuery Tasks**  
  Several tasks implemented to demonstrate DOM manipulation, events, and animations (e.g., notifications, copy-to-clipboard, lazy loading).

- **Audio and Visual Interactions**  
  Button click sound effects and hover animations on show cards.

- **Terms & Conditions Link**  
  Displayed in the footer of every page for accessibility and professionalism.

## Setup & Usage

1. Download or clone the repository:
   git clone https://github.com/Ali-Baimen/FinalWebTech.git
   cd TV-Lite
Open index.html in any modern browser (Chrome, Edge, Firefox).

Navigate through pages using the top navigation bar.

To test features:

Toggle the theme switch (top-right corner).

Use the search box on the “Shows” page.

Submit the “Support” form to see validation.

Try signing up and reviewing the form behavior.
